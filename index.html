Siklus Pengembangan Sistem Rental Mobil

SiklusAktivitasDokumen
Perencanaan Mengidentifikasi kebutuhan sistem, menentukan fitur utama seperti daftar mobil, pemesanan, dan pembayaran. Menetapkan jadwal proyek, anggaran, serta pembagian tugas antar tim pengembang. Dokumen Perencanaan Sistem (Project Plan)
Analisis Kebutuhan Mengumpulkan data dari pengguna dan admin rental mobil, mendefinisikan kebutuhan fungsional dan non-fungsional. Dokumen Analisis Kebutuhan (SRS)
Desain Mendesain database, struktur sistem, serta tampilan antarmuka (UI) setiap halaman website. Dokumen Desain Sistem (System Design Document)
Implementasi Mengembangkan website sesuai desain menggunakan HTML, CSS, PHP, dan database MySQL. Source Code dan Dokumentasi Implementasi
Testing Melakukan pengujian terhadap semua fitur, memastikan sistem berjalan sesuai kebutuhan dan bebas error. Laporan Hasil Uji (Testing Report)
Deployment Mengunggah website ke server hosting, memastikan website dapat diakses publik. Dokumen Rilis Sistem (Deployment Report)
Maintenance Melakukan pembaruan fitur, perbaikan bug, dan peningkatan performa website secara berkala. Laporan Pemeliharaan (Maintenance Report)
